## Language and frameworks
- Nodejs (12.x)
- MongoDB
- Expressjs
- Redis
## Endpoints
- Questions [/question]
  1. Post Question [/]  method => [post]
  2. Get All Questions [/] method => [get]
  3. Get Single question [/:id] method => [get]
    
- Answers [/answer]
  1. Answer Question [/question/:id] method => [post]
